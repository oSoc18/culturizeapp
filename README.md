# Resolver

This is the GitHub repo of the Resolver Project of oSoC18.

## Setup 

* Clone the repo
  * `git clone https://github.com/oSoc18/resolver.git`
* Set the working directory
  * `cd resolver`
* Run npm install
  * `npm install`
  
## Run the app

* Use npm start
  * `npm start`
